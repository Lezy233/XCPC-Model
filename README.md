# 目录介绍
- _Check
    - 用于对拍的一些基本模板（实际上我基本不用）
- Computational Geometry 计算几何
    - 难度不高，代码量比较大，同时又可能出现卡常卡精度之类的问题，在赛时的优先级比较低
    - 之前我在我们队主要负责此块
- Data Structure 数据结构
    - 重难点，主要是线段树主席树之类的
    - 也是我主要负责的地方，但是在正式比赛当中好像从来没有怎么敲出来过
- Game Theory 博弈论
    - 没有什么东西，就一个 SG 函数，赛时很少用到
- Graph Theory 图论
    - 重点，难度可高可低，其中有一个网络流这一块相当恶心
    - 我新手的时候最喜欢学这部分内容，因为用途广，性价比高
- Hash 哈希
    - 没东西，就是一个手写哈希来防止 codeforces 上被别人 hack ，但是你为什么不用 `std::map` 呢
    - 字符串哈希归类到了字符串那里
- Math 基础数学
    - 比较有用的是一个 `Mint` 自动取模模板，用得爱不释手
- Math Theory 数论
    - 难点
    - 这部分主要是我的神仙队友负责
- Other 其他
    - 各种不知道如何归类的内容就被归类到了这里
    - 基本上不怎么用到
- String 字符串
    - 重难点
    - 比较基础的也就一个字符串哈希和 KMP。实际使用下来字符串哈希能解决大部分基础的字符串问题。接下来的就是难度极高的各种自动机，说实话我也只是掌握了基本的模板
